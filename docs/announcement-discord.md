## Post (paste-ready)

**Games — an Android TV home-screen widget for Sunshine + Moonlight** 🎮📺

Hey everyone! I made a small open-source Android TV app that puts your **Sunshine** game
library right on the TV **home screen**, and launches each game through **Moonlight**.

I originally built it just **for myself**, but figured it might be useful to someone else too,
so I'm sharing it.

Full disclosure: **I'm not a programmer** — I built the whole thing with **Claude Code**. So
please be kind 😅

Features: per-game show/hide checkboxes, alphabetical sorting, custom widget name, English + Ukrainian UI.

It's **GPLv3** and builds on the Moonlight project. Big thanks to the Moonlight and Sunshine teams 🙏

**Moonlight devs — feel free to use this however you like.** And if you ever decide to build
this kind of home-screen integration into Moonlight itself, that would be awesome! 🚀

I've done everything I wanted with it, so I'm releasing the source code — anyone who's
interested is welcome to take it and adapt it to their own needs.

GitHub: https://github.com/Frostoman/games-widget-androidtv
